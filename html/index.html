Welcome to lolfu!

Want to find out which champion you should play in each position? Enter your summoner name and we'll make a recommendation that optimizes your solo queue climb.

How does this work!?

Lolfu uses science. We model champion selection as a multi-armed bandit problem where each champion is an arm. We then recommend champions using an epsilon-greedy strategy setting epsilon equal to zero.

Why epsilon zero? Because there are too many champions and not enough time within a single season to spend too much time exploring your champion pool. With 100+ champions playing 10 games on each champion would be >1000 games!
